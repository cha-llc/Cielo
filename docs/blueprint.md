# **App Name**: Cielo

## Core Features:

- AI-Powered Daily Affirmations: Generates a unique motivational and disciplined affirmation each day using an AI tool, optionally personalized by zodiac sign if the user is upgraded.
- User Authentication: Secure login/signup with email confirmation to manage user accounts.
- Mood Journal: Allows users to track their daily mood and keep a journal with sentiment analytics.
- Personalized Zodiac Affirmations: Upgraded feature that personalizes affirmations based on the user's zodiac sign.
- Subscription Management: Handles user subscriptions and unlocks premium features upon upgrade.
- Profile and Settings: Allows users to manage their profile information and app settings.

## Style Guidelines:

- Primary color: Deep navy blue (#000080) for a calming and sophisticated background.
- Accent color: Crimson red (#DC143C) for interactive elements and highlights, conveying energy and passion.
- Secondary accent: Emerald green (#50C878) to add a touch of balance and tranquility, symbolizing growth and harmony.
- Text color: Off-white (#F0F8FF) for readability against the dark background, ensuring clarity and comfort.
- Headline font: 'Oswald' sans-serif for titles, providing a modern and bold feel with excellent readability.
- Body font: 'Roboto' sans-serif for body text and UI elements, ensuring clarity and accessibility across all screen sizes.
- Use filled, vibrant icons in crimson red and emerald green to maintain a cohesive and refined theme.
- Mobile-first design with a bottom tab bar for main navigation. Use auto-layout for responsiveness across devices.
